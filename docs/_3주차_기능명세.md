# 기능 명세

## 기능
### 바우처
- [ ] 모든 바우처를 조회한다.
- [ ] 할인 정책별로 바우처들을 조회한다.
- [x] 바우처를 생성한다.
- [ ] 바우처를 삭제한다.

## 뷰
### 바우처
- [ ] 모든 바우처 조회 페이지
- [ ] 특정 바우처 상세 조회 페이지
- [ ] 바우처 생성 페이지
- [ ] 바우처 삭제 페이지
### 회원
- [ ] 회원을 조회 페이지
- [ ] 특정 회원 상세 조회 페이지
- [ ] 회원 생성 페이지
- [ ] 회원 삭제 페이지
### 지갑
- [ ] 특정 고객에게 바우처를 할당하는 페이지
- [ ] 고객의 모든 바우처를 조회하는 페이지
- [ ] 해당 바우처를 가진 특정 고객을 조회하는 페이지
- [ ] 고객이 가진 바우처를 삭제하는 페이지
